# W-DA — Study Material

A simple static website collecting unit-wise study notes for a (W-DA) course. Built with plain HTML and CSS, and published live via GitHub Pages.



## Overview

This repository serves as a personal reference/homepage that links out to individual unit pages, each covering a specific topic in web development.

## Contents

| Unit | Topic | File |
|------|-------|------|
| Unit 1 | Introduction to WWW and HTML | `unit1.html` |
| Unit 2 | About the DOM (Document Object Model) | `unit2.html` |
| Unit 3 | About CSS | `unit 3.html` |
| Unit 4 | About Blogs | `unit4.html` |
| Unit 5 | Google Analytics and Digital Marketing | `unit 5.html` |

Other files:
- `index.html` — Homepage with navigation links and thumbnail images for each unit
- `3fig.css` — Stylesheet used by supporting pages
- `1fig.html`, `2fig.html`, `3fig.html` — Supplementary/example pages
- `unit1.jpeg`, `unit2.png`, `unit3.png`, `unit5.jpeg`, `unnit4.jpeg` — Images used across the unit pages

## Tech Stack

- HTML5
- CSS3

## Getting Started

To view the site locally:

1. Clone the repository
```bash
   git clone https://github.com/saina39/W-DA.git
```
2. Open `index.html` in your browser.

No build steps, dependencies, or server setup are required — it's all static HTML/CSS.
